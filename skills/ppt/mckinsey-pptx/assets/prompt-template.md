# McKinsey PPT — Prompt 模板

## 完整 Deck 新建 Prompt

```
用 slides 和 imagegen 创建一份麦肯锡风格的咨询演示文稿。

项目背景：{一句话描述项目/客户/问题}
核心结论：{用一句话概括你的核心建议或发现}
目标受众：{C-level / 投资方 / 内部决策层}
建议页数：{8-15 页}
关键数据/报告：{如有现成报告或数据，粘贴链接或要点}

请严格遵循以下流程：
1. 先输出 SCQA 叙事框架和页面规划（每页标题即结论）
2. 确认后再开始逐页生成
3. 使用麦肯锡经典蓝配色（primary: 003c71）
4. 每页 Header 栏 + 标题区 + 内容区 + 来源/页码
5. 数据图表标注来源和年份
6. 最终交付 .pptx + 生成脚本
```

## 风格转换 Prompt

```
将以下 PPT 改为麦肯锡风格。

输入文件：{路径}
核心结论：{一句话概括}
关键数据来源：{如有}

修改要求：
1. 添加 McKinsey 风格 Header 栏（深蓝色顶栏）
2. 所有标题改为结论句（描述型→判断型）
3. 应用麦肯锡配色方案
4. 统一字体（Arial / Microsoft YaHei）
5. 每页底部添加来源标注和页码
6. 保持数据图表简洁、去掉 3D 效果
7. 如果页数超过 15 页，先输出优化建议
```

## 单页/增量 Prompt

```
在麦肯锡风格 deck 中新增一页。

位置：第{N}页之后
核心信息：{一句话结论}
数据/内容：{要点或数据}

请生成该页的 createSlide 函数代码，标准 Content 页布局，
标题即结论，标注数据来源。
```
