# Frontend Slides — Prompt 模板

## 技术演讲完整 Prompt

```
用 slides 和 imagegen 创建一份技术演讲幻灯片。

演讲主题：{一句话主题}
目标受众：{初级/中级/高级前端开发者}
演讲时长：{15 / 30 / 45 分钟}
场景：{技术大会 / 内部分享 / 培训}
核心要点：{列出 3-5 个要讲的技术点}
代码示例：{关键代码片段或仓库链接}
性能数据（如有）：{数据点}

请按以下流程执行：
1) 先输出演讲结构规划（Agenda + 每页内容概要）
2) 确认后再逐页生成
3) 选择【深色演讲】或【浅色分享】配色
4) 代码块使用 Consolas 字体，关键行高亮
5) 数据指标使用大数字展示
6) 最终输出 .pptx + 生成脚本
```

## 内部分享 Prompt

```
用 slides 和 imagegen 做一份技术内部分享的 PPT。

分享主题：{主题}
团队水平：{初级/中级/混合}
时长：{30 / 45 / 60 分钟}
想让大家带走什么：{一句话学习目标}
主要内容：{要点列表}

要求：
1) 浅色主题配色，可读性优先
2) 每个概念先用一句话大白话定义
3) 代码块不超过 20 行
4) 适当加"思考点"或互动环节
5) 最后有总结回顾页
```

## 代码对比 Prompt

```
帮我做一页代码对比的 slide。

主题：{对比内容 — 如 "优化前后性能对比"}

Before 代码：
```{语言}
{代码}
```

After 代码：
```{语言}
{代码}
```

关键数据：{如 "LCP 从 3.2s 降到 0.8s"}

请生成这页的 createSlide 函数，使用左右分栏布局，
左边 Before（红色标注），右边 After（绿色标注），
底部突出性能提升数据。
```
