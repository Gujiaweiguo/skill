# 领域标签体系

此文件定义素材分类的 `domain` 标签体系，由 material-importer 的 P2 分类映射引用。
**可按需调整**：新增/修改业务线时只需更新此文件。

## domain 标签一览

| 业务线 | domain 标签 | 说明 |
|-------|------------|------|
| 商管 | 商管 | 商业管理ERP、招商租赁、商管系统 |
| 会员 | 会员 | 会员CRM、SCRM、积分商城、私域运营 |
| AI客服 | AI客服 | AI智能客服、智能导购、AI问答 |
| AI问数 | AI问数 | AI数据分析、BI智能分析、AI决策 |
| 通用 | 通用 | 所有业务线共享的素材 |

> 一条素材可同时属于多个业务线（如"商圈会员CRM"同时服务于商管和会员，domain: ["商管", "会员"]）

## P2 分类映射表

| 内容特征 | 目标目录 | type 标签 | domain 标签 |
|---------|---------|----------|-----------|
| 公司简介、愿景、发展历程、组织架构 | 01-company-overview/ | 公司概况 | 通用 |
| 财报、业绩数据 | 01-company-overview/ | 公司概况 | 通用 |
| ISO认证、高企、双软、软著、专利、奖项 | 02-qualifications/ | 资质荣誉 | 通用 |
| 产品功能、技术方案、架构设计 | 03-products/ | 产品方案 | 商管/会员/AI客服/AI问数 |
| 客户项目、背景、需求、解决方案 | 04-cases/ | 案例 | 商管/会员/AI客服/AI问数 |
| 实施方法论、实施步骤、项目管理 | 05-implementation/ | 实施方法论 | 通用 |
| 售后运维、SLA、服务标准、培训 | 06-services/ | 服务体系 | 通用 |
| 人员简历、职称、证书、项目经验 | 07-personnel/ | 人员 | 按专长 |

## 使用说明

- 分类时按内容特征→目标目录→type→domain 逐级映射
- domain 为"通用"的素材所有业务线都可以引用
- domain 为具体业务线的素材只在对应业务线场景下匹配
- 案例类和产品类通常绑定具体业务线 domain
